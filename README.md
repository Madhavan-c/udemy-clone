🎓 Udemy Clone – HTML & CSS Frontend Project
📖 Overview
This project is a responsive frontend clone of the Udemy homepage, crafted using HTML5 and CSS3. It emulates the layout and design of the popular online learning platform, focusing on clean structure, responsive design, and modern UI components. The project serves as a demonstration of proficiency in HTML and CSS, particularly utilizing Flexbox for layout management.

🚀 Features
Responsive Layout: Adapts seamlessly to various screen sizes, ensuring usability across desktops, tablets, and mobile devices.

Flexbox Utilization: Employs CSS Flexbox for efficient and flexible layout structuring.

Course Listings: Displays courses with images, titles, and brief descriptions in a grid format.

Navigation Bar: Includes a top navigation bar for easy access to different sections.

Search Functionality: Features a search bar to simulate course search capability.

Footer Section: Contains links to social media, contact information, and additional resources.
